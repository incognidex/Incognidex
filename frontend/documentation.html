CSS selectors

CSS selectors are patterns used in CSS rules to target and select specific elements for styling.

For example, to style paragraphs, you will use the p type selector to select all <p> elements and apply a style to them:

CSS 
/* Set font size on all <p> elements */
p {
  font-size: 12px;
  color: rebeccapurple;
}

Syntax

CSS 
/* Select elements and apply styles */
selector {
  property: value;
}
